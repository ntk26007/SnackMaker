#Instalacion de la aplicacion de escritorio:
1. En el cmd poner:
npm install electron --save-dev
npm install electron-builder --save-dev
npm install cross-env --save-dev
2. Ejecutar la aplicacion
npm run electron
3. Instalar el instalador de la app:
npm run dist:win
